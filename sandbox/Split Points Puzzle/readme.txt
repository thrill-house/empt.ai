Opening the game in a browser will generate a solved level.
The start position is black, goals are red, occupied cells are blue.
The numbers appearing in the cells after a move are the minimum number of moves that were used to get to this cell.
Clicking "Clear" will reset the level to the start.
Clicking a cell in the play area will select it as start position for the next move.
Select a direction and the number of steps, then click "Move" to execute the move. 
Invalid moves will open an alert.
Clicking "Revert" will revert the last move.
